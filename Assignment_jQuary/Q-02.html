To apply css

            $('button').click(()=>{
                  $('div').css({'background-color': 'yellow', 'font-size':'1.5em'});
            })

To add class and remove class

            *********** Add class ****************

            $(".btn1").click(function(){
                  $("div").addClass("alert");
            });
            
            *********** Remove class ****************

            $(".btn2").click(function(){
                  $("div").removeClass("alert");
            });

JQuery Animation

-> The jQuery animate() method is used to create custom animations.

            $("button").click(function(){
                  $("div").animate({
                        opacity: '0.5',
                        height: '300px',
                        width: '500px'
                  });
            });
